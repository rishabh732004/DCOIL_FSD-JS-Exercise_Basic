JavaScript Debugging Exercise

Your task is to fix the function troubleshooting so that it logs the number 2 to the console.

Currently, the function is returning an incorrect result. Carefully analyze the code and make the necessary changes without hardcoding the value.

Once you think you've fixed the issue, open the browser's Developer Console (Right-click and select "Inspect" then go to the "Console" tab) to see if the number 2 is logged as the result.

Hint: The problem is in how the variables are being combined in the result line.
