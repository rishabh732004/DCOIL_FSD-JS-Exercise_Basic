JavaScript Variables Exercise

This exercise involves predicting the result of a code snippet, then recreating it after understanding how it works.

Instructions:

  • Step 1: Look at the code and make a prediction about the output in the console. Run the code and check the result.
  • Step 2: After understanding the code, delete it and recreate the variables following the instructions in the comments.

Tip: You can use console.log to check your values at any time while working through the exercise.
